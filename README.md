#### Dependencia de los siguientes paquetes

    * gnuradio          (Versión >= 3.8)
    * libboost-all-dev  (Verisón >= 1.71)
    * libcppunit-dev    (Versión >= 1.15.1)
    * liblog4cpp5-dev   (Versión >= 1.1.3)
    * liborc-0.4-dev    (Versión >= 0.4.32)
    * swig              (Versión >= 4.0.1)
    * cmake             (Versión >= 3.16.3)
    * git               (Versión >= 2.25.1)
    
#### Build instructions

    * mkdir build
    * cd build
    * cmake ..
    * make
    * sudo make install
    * sudo ldconfig
